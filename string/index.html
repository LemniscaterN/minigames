貴方の数字「　0　」です

お題 : ボタンを押してね！

String(ストリング)

    推奨人数：3人以上　プレイ時間：1ゲーム10分程度
    このゲームは主に概要のように進みます。詳しいルール説明については【詳細】以降をご覧ください。

    概要

  1. 自分の「点数」を決める。「点数」は他の人には言ってはいけない。
  2. お題を決める。
  3. お題に対して自分の「点数」があてはまりそうなものを考える。
    例）　お題：熱いもの　点数：89　→熱湯
  4. みんなで自分の考えたモノを発表し、点数が高いと思う順番に並び替える。
    例)　熱湯　>　真夏のアスファルト　>　アイス
  5. それぞれ自分の点数を発表し、数字の大きい順にならんでいたら成功です。

    詳細

  1. 各プレイヤーは「SetNumber」を1回押して1-100の数字を一つ決めます。これが貴方の点数になります。
  2. メンバーの誰かが「NormalTopic」か「RandomTopic」を押して今回のお題を決め、全員に共有します。
    • NormalTopic : それっぽいお題です。50個程度あります。
    • RandomTopic : お題を自動生成します。意味不明なお題が表示されることもあるので要注意。
  3. お題に最も適したもの100点として、自分の点数に当てはまるものを考えてみましょう。
    (つまり条件に自分が最も合うと思う言葉が100点、最も合わない言葉が1点となります)

  4. 例えばお題が「偉そうな人」の場合、以下のようになります。
    例えば数字が「92」なら、僕のイメージはエジソンです。

    • 100点：超絶偉そうな人
    • 1点：全く偉く無さそうな人

  5. それぞれが自分の点数だと思う言葉を発表し、話し合います。この時、自分の引いた点数を口にしてはいけません。
    その後、数字の小さい順（or大きい順）に公開出来れば成功です！
